The updated version of ggplot2 V 2.0 has improved the way we can label panels in facet plots with the use of a generic labeller function. The latex2exp package has made it much easier to write expressions in R.

You will need to load the following packages for the code below to work:

  1. devtools
  2. ggplot2
  3. latex2exp
